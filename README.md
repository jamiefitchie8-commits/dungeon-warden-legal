# Dungeon Warden Legal Pages

Public legal and support pages for the Dungeon Warden Alexa skill by The Sherbet Lemon.

This repository intentionally contains only static legal/support pages and the public GitHub feedback form. It does not contain game source code, AWS configuration, saves, transcripts, secrets, or private deployment material.
